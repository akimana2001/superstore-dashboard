# superstore-dashboard
 BSA83111 – Data Visualization for Decision Making | Kepler College | 2026
This dashboard uses Python and Streamlit to analyze the Sample Superstore dataset and visualize key business insights including sales trends, profitability, regional comparisons, and ethical considerations. The dataset was sourced from Tableau Software at https://www.tableau.com/learn/articles/sample-data-sets
To run: Install dependencies with pip install streamlit plotly pandas openpyxl then run streamlit run Superstore_Dashboard_Final.py
